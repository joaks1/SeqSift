Version 0.1.2
=============

New Features
------------
  - ``aligndicer.py`` tool for extracting columns form aligned sequences.
  - API support for PyMsBayes; can automate preparing of input files.

    are in path, rather than requiring full paths, for aligner-tool options.

Changes
-------
  - New functions added for checking and handling paths to external tools.
